# 🎣 낚고알고 (Nakgo-Algo)

## 📌 프로젝트 소개

**낚고알고**는 낚시인을 위한 규제 정보 통합 서비스입니다.

금지구역, 금어기, 체장 규정 등 복잡한 낚시 규제를 한눈에 확인하고, AI 어종 분석까지 제공하여 안전하고 합법적인 낚시를 돕습니다.

---

## ❓ 왜 만들었는가?

직접 낚시를 하면서 겪은 불편함에서 출발한 프로젝트입니다.

### 1. 규제 정보를 찾기가 너무 어려웠다
- 낚시 현장에서 "여기서 낚시해도 되나?", "이 물고기 가져가도 되나?" 궁금할 때 바로 확인할 수 있는 방법이 없었습니다.
- 규제 정보가 수산자원관리법, 물환경보전법 등 여러 법령에 흩어져 있어 일일이 찾아봐야 했습니다.

### 2. 규정을 몰라서 위반할 뻔했다
- 금지체장, 금어기 같은 규정을 현장에서 바로 확인할 수 없어 모르고 위반할 뻔한 경험이 있었습니다.
- 어종과 크기만 입력하면 즉시 위반 여부를 알려주는 기능이 있으면 좋겠다고 느꼈습니다.

### 3. 잡은 물고기가 뭔지 몰랐다
- 물고기를 잡았는데 정확한 어종을 몰라 규제 확인 자체가 불가능한 경우가 있었습니다.
- 사진만 찍으면 AI가 어종을 알려주고, 해당 어종의 규제 정보까지 바로 확인할 수 있도록 개발하였습니다.

## 🛠 주요 기능

| 기능 | 설명 |
|------|------|
| 🗺 **금지구역 지도** | 전국 낚시금지구역을 지도와 목록으로 조회 |
| ⚖️ **위반 판단** | 어종 + 크기 입력으로 포획 가능 여부 즉시 확인 |
| 📷 **AI 어종 인식** | 사진 촬영/업로드로 어종 추정 및 규제 안내 |
| 📋 **규제 정보** | 금어기, 금지체장 등 어종별 규제 정보 제공 |
| 💰 **벌금 안내** | 위반 시 과태료 정보 안내 |
| 🚨 **오류 제보** | 사용자가 잘못된 정보를 신고하여 데이터 정확성 향상 |
