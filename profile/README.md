# 🎣 낚고알고 (Nakgo-Algo)

## 📌 프로젝트 소개

**낚고알고**는 낚시인을 위한 규제 정보 통합 서비스입니다.

금지구역, 금어기, 체장 규정 등 복잡한 낚시 규제를 한눈에 확인하고, AI 어종 분석까지 제공하여 안전하고 합법적인 낚시를 돕습니다.

---

## ❓ 왜 만들었는가?

### 1. 흩어진 낚시 규제 정보
- 낚시 관련 규제는 수산자원관리법, 물환경보전법 등 여러 법령에 분산되어 있습니다.
- 일반 낚시인이 현장에서 빠르게 확인하기 어려운 구조입니다.

### 2. 모르고 위반하는 경우 방지
- 금지체장, 금어기 등 규정을 모르고 위반하여 과태료를 부과받는 사례가 많습니다.
- **낚고알고**는 어종과 크기만 입력하면 즉시 위반 여부를 판정합니다.

### 3. AI를 활용한 어종 인식
- 잡은 물고기 사진을 찍으면 AI가 어종을 추정하고 관련 규제 정보를 자동으로 안내합니다.

---

## 🛠 주요 기능

| 기능 | 설명 |
|------|------|
| 🗺 **금지구역 지도** | 전국 낚시금지구역을 지도와 목록으로 조회 |
| ⚖️ **위반 판단** | 어종 + 크기 입력으로 포획 가능 여부 즉시 확인 |
| 📷 **AI 어종 인식** | 사진 촬영/업로드로 어종 추정 및 규제 안내 |
| 📋 **규제 정보** | 금어기, 금지체장 등 어종별 규제 정보 제공 |
| 💰 **벌금 안내** | 위반 시 과태료 정보 안내 |
| 🚨 **오류 제보** | 사용자가 잘못된 정보를 신고하여 데이터 정확성 향상 |
