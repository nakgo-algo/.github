# 🎣 낚고알고 (Nakgo-Algo)

## 📌 프로젝트 소개

**낚고알고**는 낚시인을 위한 규제 정보 통합 서비스입니다.

금지구역, 금어기, 체장 규정 등 복잡한 낚시 규제를 한눈에 확인하고, AI 어종 분석까지 제공하여 안전하고 합법적인 낚시를 돕습니다.

---

## ❓ 왜 만들었는가?

### 실제 낚시를 하며 겪은 어려움

- **규제 정보를 확인할 수 없는 상황** — 낚시 현장에서 "여기서 잡아도 되나?" 싶을 때, 관련 규정이 수산자원관리법, 물환경보전법 등 여러 법령에 흩어져 있어 바로 찾아볼 수 없는 상황.
- **규정을 모르고 위반하게 되는 상황** — 금지체장, 금어기 같은 규정을 현장에서 확인할 방법이 없어 자신도 모르게 위반하게 되는 상황.
- **잡은 물고기의 어종을 모르는 상황** — 물고기를 잡았지만 정확한 어종을 몰라 규제 확인 자체가 불가능한 상황. 사진만 찍으면 AI가 어종을 판별해주는 기능을 통해 해결.

---

## 🛠 주요 기능

| 기능 | 설명 |
|------|------|
| 🗺 **금지구역 지도** | 전국 낚시금지구역을 지도와 목록으로 조회 |
| ⚖️ **위반 판단** | 어종 + 크기 입력으로 포획 가능 여부 즉시 확인 |
| 📷 **AI 어종 인식** | 사진 촬영/업로드로 어종 추정 및 규제 안내 |
| 📋 **규제 정보** | 금어기, 금지체장 등 어종별 규제 정보 제공 |
| 💰 **벌금 안내** | 위반 시 과태료 정보 안내 |
| 🚨 **오류 제보** | 사용자가 잘못된 정보를 신고하여 데이터 정확성 향상 |
